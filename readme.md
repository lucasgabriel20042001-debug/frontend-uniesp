# 🛥️ Luxo — Aluguel de Iates

Atividade prática da disciplina de **Front-End** do curso de graduação da **UNIESP**.

O projeto consiste em uma landing page fictícia para uma empresa de aluguel de iates de luxo, desenvolvida do zero com **HTML5** e **CSS3 puros**, sem o uso de frameworks ou bibliotecas externas.

---

## 📄 Sobre o Projeto

A página apresenta:

- **Header fixo** com logo em SVG, nome da empresa e menu de navegação
- **Seção Hero** com imagem de fundo, overlay gradiente e card de destaque com chamada para ação
- **Grade de conteúdo** dividida em duas seções: *Destinos* e *Nossa Frota*, com imagens e textos
- **Design responsivo** adaptado para dispositivos móveis
- **Footer** com créditos da empresa

## 🚀 Como Rodar o Projeto

Não é necessário instalar nada. Siga os passos abaixo:

### Opção 1 — Baixar o ZIP (mais fácil)

1. Acesse o repositório: [github.com/lucasgabriel20042001-debug/frontend-uniesp](https://github.com/lucasgabriel20042001-debug/frontend-uniesp)
2. Clique no botão verde **`<> Code`**
3. Selecione **Download ZIP**
4. Extraia o arquivo baixado
5. Abra a pasta extraída e dê um duplo clique em **`index.html`**
6. A página abrirá automaticamente no seu navegador padrão ✅

### Opção 2 — Clonar com Git

Se você tem o Git instalado, abra o terminal e execute:

```bash
git clone https://github.com/lucasgabriel20042001-debug/frontend-uniesp.git
cd frontend-uniesp
```

Em seguida, abra o arquivo `index.html` no navegador.
